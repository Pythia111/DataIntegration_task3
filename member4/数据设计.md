下面为 formatStudent.xsd 的示例：

```xml
<?xml version= "1.0" encoding= "gb2312"?>
<xsl:stylesheet version= "1.0" xmlns:xsl= "http://www.w3.org/1999/XSL/Transform">
<xsl:output method= "xml" encoding= "gb2312"/>
<xsl:template match="Students">
<xsl:apply-templates/>
<Students>
<xsl:for-each select="student">
<student>
<xsl:choose>
<xsl:when test="A 条件">
<id>
<xsl:value-of select="*****"/>
</id>
</xsl:when>
<xsl:when test=" B 条件'">
<id>
<xsl:value-of select="*****"/>
</id>
</xsl:when>
<xsl:when test="C 条件">
<id>
<xsl:value-of select="*****"/>
</id>
</xsl:when>
</xsl:choose>
<name>
<xsl:value-of select="姓名"/>
<xsl:value-of select="Snm"/>
</name>
<sex>
<xsl:value-of select="性别"/>
<xsl:value-of select="Sex"/>
</sex>
<xsl:choose>
<major>
<xsl:value-of select="专业"/>
</major>
</xsl:choose>
</student>
</xsl:for-each>
</Students>
</xsl:template>
</xsl:stylesheet>
```



### 2.2 XSL 定义

所有文件都部署在集成服务器上，所需文件和作用如下表格所示：

表 3-16：XSL 文件部署

| 文件名称              | 作用                             |
| --------------------- | -------------------------------- |
| formatClass.xsl       | 各系统课程格式转换到统一课程格式 |
| formatStudent.xsl     | 各系统学生格式转换到统一学生格式 |
| formatClassChoice.xsl | 各系统选课格式转换到统一选课格式 |
| studentToA.xsl        | 统一学生格式转换到 A 学生格式    |
| studentToB.xsl        | 统一学生格式转换到 B 学生格式    |
| studentToC.xsl        | 统一学生格式转换到 C 学生格式    |
| classToA.xsl          | 统一课程格式转换到 A 课程格式    |
| classToB.xsl          | 统一课程格式转换到 B 课程格式    |
| classToC.xsl          | 统一课程格式转换到 C 课程格式    |
| choiceToA.xsl         | 统一选课格式转换到 A 选课格式    |
| choiceToB.xsl         | 统一选课格式转换到 B 选课格式    |
| choiceToC.xsl         | 统一选课格式转换到 C 选课格式    |



都部署在集成服务器端，用来转换各个院系教务系统发过来的学生，课程，选课 XML 数据到统一格式的 XML，再将统一格式的 XML 转换成目标院系教务系统的期望 XML 格式。
